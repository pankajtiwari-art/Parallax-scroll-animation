A stunning, highly responsive parallax scrolling experience built to add depth and visual dynamism to web interfaces. This project demonstrates modern front-end design techniques to create an immersive user journey through seamless scroll-triggered animations.

## 🚀 Live Demo

Check out the live action here: **[View Live Demo](https://pankajtiwari-art.github.io/Parallax-scroll-animation/)**

## 🌟 Key Features

* **Smooth Parallax Effect:** Multi-layered backgrounds scrolling at different speeds to create a captivating 3D illusion.
* **Fully Responsive:** Perfectly adapts to all screen sizes—from large desktop monitors to mobile devices.
* **Lightweight & Optimized:** Built with pure code (Vanilla JS)—no heavy external libraries or dependencies required.
* **Modern Aesthetics:** Clean UI/UX designed to capture user attention and enhance engagement.

## 🛠️ Technologies Used

* **HTML5:** Clean, semantic markup.
* **CSS3:** Advanced styling, absolute positioning, and smooth transitions.
* **JavaScript (ES6+):** Efficient scroll event listeners and dynamic DOM manipulation.
